Pet Health & Welfare Database - SQL

1. To create and populate the database:
   - Open schema_and_data.sql in MySQL Workbench
   - Run the entire script
   - This creates the schema (tables) and inserts sample data
   - Database name: pet_welfare_test (or pet_welfare)

2. To run the queries:
   - Open queries.sql in MySQL Workbench
   - Make sure you're using the pet_welfare_test schema
   - Run individual SELECT / INSERT / UPDATE / DELETE statements as needed

Notes:
- All SQL was tested locally in MySQL Workbench.
- CRUD examples (insert/update/delete) are included near the bottom of queries.sql.
